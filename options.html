翻译助手 设置

基本设置

开启后，每次打开新页面会自动翻译。也可用 Alt+A 快捷切换

翻译通道管理

启用或停用翻译通道，停用的通道不会出现在引擎选择列表中

    自定义大模型 API

    填写 OpenAI 兼容接口地址，会自动补全 /chat/completions
    使用 {targetLang} 作为目标语言占位符

    快捷键

    Alt+T — 翻译当前页面
    Alt+S — 翻译选中文本
    Alt+R — 切换显示原文/译文（翻译后可用）
    Alt+A — 开关自动翻译新页面

    Alt+T 和 Alt+S 可在 Chrome 扩展快捷键设置页面修改：
    chrome://extensions/shortcuts
