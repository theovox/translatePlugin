# 翻译助手 - Chrome 翻译插件

> 100% AI Coding - 本项目完全由 AI（Claude Opus 4.7）编写，包括所有代码、架构设计和功能实现，未手写一行代码。

一款轻量级 Chrome 浏览器翻译插件，支持整页翻译、选中翻译，内置多个免费翻译通道，同时支持接入自定义大模型 API 进行高质量翻译。

## 功能特色

### 多种翻译模式
- **整页翻译** - 一键翻译页面所有可见文本
- **选中翻译** - 选中文本后通过快捷键或右键菜单翻译，结果以浮动面板展示
- **输入翻译** - 在插件弹窗中直接输入文本翻译
- **双语对照** - 原文与译文同时显示，方便对照阅读
- **替换原文** - 直接用译文替换原文，阅读更沉浸

### 多翻译通道
| 通道 | 特点 | 费用 |
|------|------|------|
| 谷歌翻译 | 速度快、语种全，推荐首选 | 免费无限 |
| Bing 翻译 | 微软翻译，质量高，学术文档友好 | 免费无限 |
| 腾讯翻译 | 国内通道，速度快，中英互译质量好 | 免费无限 |
| 自定义大模型 | 翻译质量最高，适合精翻 | 按 token 计费 |

- 通道可在设置页自由启用/停用
- 自定义大模型支持所有 OpenAI 兼容接口（GPT、Claude、DeepSeek 等）

### 智能特性
- **自动翻译** - 开启后自动翻译新打开的页面，中文页面自动跳过
- **SPA 页面支持** - 监听 URL 变化（pushState/replaceState/popstate），文档类页面切换路由时自动翻译
- **翻译中断** - 页面变化或重新翻译时，自动中断进行中的翻译任务
- **二次翻译** - 已翻译的页面可以重新翻译（切换引擎后直接翻译，无需刷新）
- **原文/译文切换** - 翻译后可随时切换显示原文或译文，不重新请求翻译

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+T` | 翻译当前页面 |
| `Alt+S` | 翻译选中文本 |
| `Alt+R` | 切换显示原文/译文 |
| `Alt+A` | 开关自动翻译 |

`Alt+T` 和 `Alt+S` 可在 `chrome://extensions/shortcuts` 自定义修改。

## 安装方式

### 开发者模式安装
1. 下载或克隆本仓库
   ```bash
   git clone https://github.com/theovox/translatePlugin.git
   ```
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

### 直接下载
1. 在 [Releases](https://github.com/theovox/translatePlugin/releases) 页面下载最新版本的 zip 文件
2. 解压后按上述步骤加载

## 使用说明

### 基本使用
1. 安装插件后，点击浏览器工具栏的插件图标打开弹窗
2. 选择翻译引擎和目标语言
3. 点击「翻译整页」或使用快捷键 `Alt+T` 翻译当前页面
4. 选中页面文本后按 `Alt+S` 可翻译选中内容

### 配置自定义大模型
1. 点击弹窗右上角「设置」进入设置页
2. 翻译引擎选择「自定义大模型」
3. 填写 API 地址（如 `https://api.openai.com/v1`）、API 密钥和模型名称
4. 可自定义系统提示词，使用 `{targetLang}` 作为目标语言占位符
5. 点击「测试连接」验证配置是否正确
6. 点击「保存设置」

### 自动翻译
1. 在设置页勾选「自动翻译新页面」，或按 `Alt+A` 快捷开关
2. 开启后，打开非中文页面会自动翻译
3. SPA 类页面（如文档站点）切换路由时也会自动翻译

### 通道管理
1. 在设置页「翻译通道管理」区域，可以启用或停用翻译通道
2. 停用的通道不会出现在引擎选择列表中

- 无需构建工具，开箱即用
